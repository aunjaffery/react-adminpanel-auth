# react-adminpanel-auth

## Minimal Admin panel skeleton for reactjs, with @chakra-ui, @tanstack/react-query, zustand and JWT auth

### Dependencies

- react 18.2
  
- @chakra-ui/react 2.8.2
  
- @tanstack/react-query 5.15.3
  
- axios 0.27.2
  
- react-icons 4.12.0
  
- react-toastify 9.1.3
  
- zustand 4.1.1
  
- jwt-decode 3.1.2
